# Excalifont Setup

To use Excalifont in this application:

1. Download Excalifont from: https://plus.excalidraw.com/excalifont/
2. Place the font files (Excalifont.woff2, Excalifont.woff, or Excalifont.ttf) in this directory
3. The font will automatically be loaded by the application

The font files should be named:
- Excalifont.woff2 (preferred)
- Excalifont.woff (fallback)
- Excalifont.ttf (fallback)
